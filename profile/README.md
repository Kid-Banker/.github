# Kid Banker - Capstone Dicoding 2026

Kid Banker is a web-based application developed as a smart solution to improve financial literacy among the younger generation while mitigating the risk of uncontrolled pay later use. Through an integrated income and expense recording system, this project positions itself as a financial education tool that bridges children's independence with transparent parental supervision.

The main output of this project is a functional financial management platform equipped with a Parental Monitoring feature to track transactions, a dedicated parental approval system for recording pay later bills, and automatic synchronization with Google Calendar as payment reminders. The end result is expected to create a healthier financial ecosystem for young people through disciplined record-keeping and measurable parental control.
